Set-Up

Download and install Arduino

  1. Run the RGBLamp.ino firmware with Arduino

  2. You’ll need to install the Adafruit Neopixel library:

      Sketch/Include a library/Manage libraries

  3. Search for «Adafruit Neopixel» in the search tab and the install latest version
  4. Upload the program on your board

RGB-Lamp-Demonstrator/02_ArduinoFirmware/RGBLamp/

Download and install Processing

  1. Run the RGBLamp.pde file with Processing while the lamp is plugged on a USB port of your computer

  2. You’ll need to install the oscP5 library:

      Sketch/Import library.../Add a library...

  3. Search for «oscP5» in the search tab and the install latest version
  4. Go to line 40 into the Processing code and set the USB port where your Arduino is plugged

RGB-Lamp-Demonstrator/03_ProcessingApplication/RGBLamp/

Launch Movuina and set-up by following the Quick Start tutorial
